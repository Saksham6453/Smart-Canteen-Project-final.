# 🍽️ Smart Canteen Pre-Order System – K.R. Mangalam University

A smart and efficient **web-based canteen management system** designed for **K.R. Mangalam University** that allows students to **pre-order meals**, make **online payments**, and track their order status in real time.
This system helps reduce long queues, saves students' break time, and improves canteen operations.

---

## 🚀 Features

### 👨‍🎓 Student Panel

* Secure Login Authentication
* Browse food menu with categories
* Add items to cart
* View item descriptions and prices
* Online Payment Gateway Simulation
* Place orders online
* Track order status in real-time:

  * Pending
  * Preparing
  * Ready
  * Completed
* View order history

---

### 👨‍🍳 Admin / Staff Panel

* Secure Admin Login
* View all student orders
* Update order status
* Manage menu stock availability:

  * In Stock
  * Out of Stock
* Manage inventory visibility
* Track sales/orders efficiently
